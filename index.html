How to add a new certificate (GitHub Pages)

  1. Create a GitHub repository (public) named yourusername.github.io or any repo and enable GitHub Pages from the main branch.
  2. Copy these files into the repo root: index.html, styles.css, app.js, certs.json, and upload certificate images into a folder named certs.
  3. Edit certs.json to add entries for each certificate (see sample below).
  4. Commit & push. Your site will be live at https://yourusername.github.io/ within a minute.

Sample certs.json entry

{
  "title": "MongoDB Basics",
  "provider": "MongoDB University",
  "year": 2024,
  "file": "certs/mongodb-basics.pdf",
  "thumb": "certs/mongodb-basics-thumb.jpg"
}

Tip: keep file names short and lowercase, and include thumbnails for faster browsing.
